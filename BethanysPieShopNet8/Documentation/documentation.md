### Architecture	
The Bethany's Pie Shop application is built using .NET 8, following a layered architecture pattern. The main components include:	
- **Presentation Layer**: Handles user interface and user experience.
- **Business Logic Layer**: Contains the core application logic and business rules.
- **Data Access Layer**: Manages data storage and retrieval, interacting with the database.
- **Database Layer**: The underlying database that stores application data.

### Features
- User authentication and authorization
- Product catalog management
- Shopping cart functionality
- Order processing and management
- Customer reviews and ratings
